# View Card Details

Access your card number, expiry date, and CVC code with ease.

## Steps

1. Open the app to land on the **Account Summary** page.
2. Scroll down to the **Card Details** section.
3. View your card number, expiry date, and CVC displayed securely.
4. Use this information as needed.

## Important Information

- The card details are currently displayed using placeholder values.
- For security reasons, in a real app, sensitive information like CVC should be handled carefully.
